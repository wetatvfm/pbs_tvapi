PBS TV Schedule API
================

== Introduction ==
A Drupal 7 module that implements the PBS TV Schedule API.

== Installation ==
Install as any other module.

== Configuration ==
Grant permission "Add and update PBS Schedule key and settings" to the user
roles who should be allowed to edit the API settings.

Next, go to admin/config/services/tvschedule_api to set the API key and Call Sign. If you do not have a key see:
https://projects.pbs.org/confluence/display/tvsapi/TV+Schedules+Version+2
